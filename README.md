# ⚓ TerminalMoses 
Parting the sea of logs for developers who hate scrolling

### 🚀 Install in one second:
`curl -sL https://raw.githubusercontent.com/brightyorcerf/terminalMoses/main/install.sh | bash`

### 💡 How to "Part the Sea" (Jump)
Since terminals are a stream of text, searching is the fastest way to travel.
Hit **Cmd+F** and search for `┏━` to instantly jump to the start of your previous commands.

### 🎨 Custom Themes
Open `~/moses.zsh` and change the `MOSES_STAFF` variable:
- **Hacker:** `green`
- **Synthwave:** `magenta`
- **Minimalist:** `242`
